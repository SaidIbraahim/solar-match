# Scripts Directory

Helper scripts that support deployment, seeding, and DevOps tasks.

## Available scripts

| Script | Description | How to run |
| --- | --- | --- |
| `backend/scripts/seedSupabase.ts` | Seeds Supabase tables with appliance presets and retailer records. | `npm --prefix backend run seed` |

You can add additional scripts (for example deployment helpers or data migrations) in this folder to keep the project root tidy. Document each script in this file for quick onboarding.
